## 使用

### 运行环境
项目对于MFC的编译方式默认使用了`在共享 DLL 中使用 MFC`，此种方式编译下的程序运行需要有相关dll库才可正常运行。  
发布版也提供了以`在静态库中使用 MFC`方式编译的程序(无-DLL结尾的.exe)，将MFC库包含程序中，可在无有关库的环境下运行。

### 使用方法
双击运行，点击按钮互动即可。
